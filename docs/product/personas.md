# 페르소나 (Personas)

## 1. 고객 (Customer / 회사 대표)
- 누구: 현재 사업을 가진 사업자. Atlas를 **AI 공동창업자**로 만난다(개정 #004).
- 첫 경험: 회원가입 = **무료 AI 컨설팅** → AI가 사업을 진단·회사를 설계 → 대표는 **승인만**.
- 운영 경험: 복잡함 없이 일이 *처리*되는 느낌. 내 회사·직원·업무 위임·결과·크레딧.
- 화면: **Customer Portal** (단순). 진단/설계 복잡성·부서/Skill OS는 비노출.
- 여정: [Customer Journey Spec](../specs/customer-journey-spec.md).

## 2. 운영자 (Operator @ Operator HQ)
- 누구: 직원을 일일이 관리하는 사람이 **아니다.** **회사 전체(조직)를 운영**하는 본사(HQ) 주체.
- 책임: **Company · Department · Employee · Skill · Training · Certification · Research Lab** 운영(개정 #002 §5).
- 원하는 것: 통제와 가시성 — Company/Department Health, KPI, 적합도(사유), 라이프사이클, ROI·원가.
- 화면: **Operator Console** (복잡 수용). → 상세: [Operator HQ Spec](../specs/operator-hq-spec.md).

## 3. AI 직원 (AI Employee) — 중심 객체(1급 시민)
- 사람이 아니지만 시스템의 핵심 행위자이자 **제품이 제공하는 가치 그 자체**.
- 독립 객체: DNA + Skill Library + Brand Memory + Training History + Certification + Performance + Matching Profile.
- → 상세: [Employee DNA Spec](../specs/employee-dna-spec.md).
